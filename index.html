Do LLMs recognise fonts?                            
                                                                                
This is a live benchmark of various LLMs' ability to recognise fonts from       
images. The latter are scraped from dafont.com. Each LLM is asked to identify   
the font whenever a new image is uploaded by a user. The guesses are compared   
against the actual font, once it has been identified by someone and confirmed.  
This process avoids benchmark contamination.                                    
                                                                                
The source code for this benchmark is available on GitHub.
